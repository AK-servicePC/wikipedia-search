Wikipedia Search settings

Language

This is the language used for all searches. Wikipedia Search will try to auto-detect your system language, but if it fails (or you just want to change it), you can set a different one below.

Other settings

Getting Started

To search, just type 'wiki' in the address bar, then a space, then the name of the article you want. Live suggestions from Wikipedia are displayed as you type, and in your selected language.

In addition, you can select any word or phrase on the web, right-click, and search it in Wikipedia instantly. Also, if you use Awesome New Tab Page, Wikipedia Search includes a widget you can add.

Having issues?

If you find a bug, or have a suggestion for a new feature, please create an issue at the Wikipedia Search GitHub repository. You can also send me a tweet or email me.
