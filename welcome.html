Getting Started

To search just type 'wiki' in Chrome's searchbar, then a space, then the name of the article you want. Live suggestions from Wikipedia are automatically displayed as you type, and in the language you pick!

In addition, you can now right-click on any word or phrase on the web and search it in Wikipedia instantly.

It's easy to change the language Wikipedia Search uses. Just go to the Settings to change to any of the 50+ languages Wikipedia supports:

Go to Settings

Don't like the update?

Wikipedia Search works differently from older versions, and we fully understand that some people might like the search button better. That's why we took the older version, added some minor improvements from newer updates, and named it Wikipedia Search Classic.

You can get Wikipedia Search Classic on the Chrome Web Store. You can even have it installed alongside the normal Wikipedia Search.

Additionally, if you have any problems with Wikipedia Search we would love it if you submitted a support request at the link below, so we can reply to you and help in any way we can.
